## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# run server
$ ng serve


```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

# settings app

```bash

# generate model
$ nest g module <name-model>

# generate controller
$ nest g controller product

# generate service
$ nest g service product

# connect nest with mongo db
$ npm i @nestjs/mongoose mongoose

# see database
$ npm install nestjs-seeder --save-dev

# installer faker
npm i faker @types/faker

```
